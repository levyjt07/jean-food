# 🍳 SmartFridge AI - Premium Edition v2.5

A futuristic, data-driven web application that analyzes refrigerator contents using the **Gemini 1.5 Flash API** and dynamically generates tailored recipes categorized by international cuisines. Built with a sleek, premium Black & Gold UI.

## ✨ Features
* **AI Image Recognition:** Analyzes your fridge contents from a single photo.
* **Cuisine Categorization:** Dynamically groups recipes into distinct culinary boxes (e.g., Italian, Indonesian, Chinese, Mexican).
* **Responsive Grid UI:** Clean, scannable card layout built with Tailwind CSS.
* **Data-Driven Architecture:** Uses strict JSON structure communication between frontend and backend.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS v4, Google Fonts (Space Grotesk)
* **Backend:** Node.js, Express.js, CORS
* **AI Engine:** Google Generative AI (Gemini 1.5 Flash)
